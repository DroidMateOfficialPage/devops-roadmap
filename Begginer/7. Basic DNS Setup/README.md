# DNS Setup Project

## Objective
Learn how to connect a custom domain to a GitHub Pages site and a static server.

---

## Task 1: GitHub Pages

- Created GitHub Pages site
- Added custom domain in repository settings
- Configured CNAME record in DNS provider
- Domain now points to GitHub Pages site

---

## Task 2: Server (Static Site)

- Created A record pointing domain to server IP
- Configured Nginx to serve static site
- Verified domain resolves to server

---

## Result
Custom domain successfully connected to both GitHub Pages and a static server.
