# Static Site Server (Local Nginx Setup)

## Overview
This project demonstrates a basic static site being served using Nginx on a local machine, simulating a remote Linux server environment.

## What was done

- Installed Nginx via Homebrew
- Configured Nginx to serve a static HTML site
- Created a simple index.html page
- Accessed the site via http://localhost:8080
- (Optional) Used rsync to simulate deployment workflow

## How it works

Browser → localhost:8080 → Nginx → static folder → index.html

## Result
A working static website served through a local Nginx web server.
